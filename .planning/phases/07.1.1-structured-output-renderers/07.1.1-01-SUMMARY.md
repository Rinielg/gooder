## Decisions Made

- Renderer subdirectory pattern: five renderers in `renderers/` folder next to `output-card.tsx` — keeps chat directory navigable, avoids one giant file
- `ChannelGroup.firstIndex` tracks position in original `output.channels` array — allows `onAdjust(channelIndex)` to pass the correct index back to page.tsx for `AdjustTarget` construction
- Multi-tier Tabs omit tier badge from card header — redundant when tab triggers show tier labels
- framer-motion staggered animation at orchestrator level only — sub-renderers do not add their own entrance animations
